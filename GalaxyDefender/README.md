# 🚀 Galaxy Defender

A space shooter game made with Python and Pygame.

## 🎮 How to Play
- ⬅️➡️ Move left and right  
- ⬆️⬇️ Move up and down  
- SPACE = shoot  
- R = reload bullets

## 🎯 Goal
Destroy UFOs, avoid asteroids, and survive!

## ✨ Features
- Shooting system 🔫  
- Reload system 🔄  
- Score system 🏆  
- Life system ❤️  
- Sound effects & music 🎵  

## 💻 Built With
- Python
- Pygame

Made while learning coding 😊
