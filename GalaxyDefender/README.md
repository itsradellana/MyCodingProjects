## ♡ **controls & mechanics**

<br>

| ♡ key | ♡ action |
|---|---|
| `←` `→` `↑` `↓` | move your ship |
| `space` | shoot laser bullets 🔫 |
| `r` | reload ammo (3 sec cooldown) 🔄 |

<br>

### ♡ game rules

```
 ✦  destroy 10 UFOs → YOU WIN! 🏆
 ✦  let 3 UFOs escape → GAME OVER 💫
 ✦  hit by UFO or asteroid → lose a life 💔
 ✦  reach 0 lives → GAME OVER 💫
 ✦  keep an eye on your ammo! only 7 bullets before reload~
```
