## 🕹️ Controls

| Key | Action |
|-----|--------|
| ⬅️ ➡️ | Move left / right |
| ⬆️ ⬇️ | Move up / down |
| `SPACE` | Shoot bullets 🔫 |
| `R` | Reload bullets 🔄 |

---

## 🎯 Goal

Destroy UFOs, avoid asteroids, and survive as long as you can!

- Reach **10 points** to **WIN** 🏆
- Don't let **3 UFOs pass** or you **LOSE** 💀
- Watch your **Life** and **Ammo**!

---

## ✨ Features

- ✅ Smooth shooting system
- ✅ Reload mechanic with cooldown
- ✅ Score, Life & Ammo tracking
- ✅ Sound effects & background music
- ✅ Asteroid obstacles
- ✅ Win / Lose conditions

## 🚀 Run Locally

Want to run the original Python version on your computer?

```bash
# Clone the repo
git clone https://github.com/itsradellana/GalaxyDefender.git
cd GalaxyDefender

# Install pygame
pip install pygame

# Run the game
python shooter_game.py
```
